# Calculator

A simple and interactive **web-based calculator** built using **HTML**, **CSS**, and **JavaScript**. This calculator allows users to perform basic arithmetic operations with a clean and responsive interface.

---

##  Features

* Basic arithmetic: **Addition (+), Subtraction (−), Multiplication (×), Division (÷)**
* **Percentage (%)** calculation
* **Clear (AC)** button to reset inputs
* Responsive and user-friendly interface
* Instant calculation display as you type/click

---

##  Project Structure

```
calculator/
├── index.html      # Main HTML page with calculator layout
├── style.css       # Styling for buttons and layout
└── script.js       # JavaScript logic for calculator functionality
```
